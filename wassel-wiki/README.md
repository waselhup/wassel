# Wassel Wiki
A compounding knowledge base maintained by Claude.

## Daily usage
- Drop anything useful into raw/ (screenshot, note, link, quote)
- Run `/wiki-ingest <filename>` to file it
- Ask questions with `/wiki-query "<question>"`
- Run `/wiki-lint` weekly

## Files you edit manually
- wassel-wiki/CLAUDE.md (only when schema rules change)
- Nothing else in wiki/ — that's Claude's domain
