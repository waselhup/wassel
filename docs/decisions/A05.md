# A05 — Every empty state has a single suggested next action

**Date:** 2026-05-25
**Status:** Accepted
**Rule:** R05

## Context

Pre-Copilot empty states read "لا توجد تحليلات" with nothing else. Users hit a dead end.

## Decision

Every empty state in Wassel includes:
1. An icon (illustrative, not decorative)
2. A short headline ("ابدأ بأول تحليل رادار")
3. A primary CTA button leading to the single next move
4. *(optional)* A secondary "Why this matters" link to a short explanation

The CTA points to the canonical next step for that surface (Radar → "ابدأ تحليلاً جديداً", Resume → "ابدأ سيرة لدور مستهدف", Content → "اقترح موضوعاً للمنشور").

## Consequences

- Every list screen has an `EmptyState` component variant for itself
- The component takes `headline`, `cta`, optional `learnMore` props
- `t('empty.radar.cta')` etc. are required keys

## Alternatives Rejected

- Text-only empty states — banned by R05
- Generic "Get started" CTA — rejected, doesn't tell the user what they're starting
