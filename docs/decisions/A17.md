# A17 — The free plan is one full lifetime journey

**Date:** 2026-05-25
**Status:** Accepted
**Rule:** R17

## Context

Most SaaS free plans give a teaser of one feature (e.g. "first analysis free"). Users never experience the full product before deciding to pay.

## Decision

The "استكشف" (Explore) plan grants enough tokens for **one complete career journey**, end to end:
- 1 Radar analysis (149)
- 1 Resume build (179)
- A few content posts (5 × 3)
- = ~343 tokens, granted as `wallet_bonus`

This is a **lifetime** entitlement (1 per user, ever — not per month). After using it, the user upgrades or stops.

**Migration to existing users (decided 2026-05-25, Q3 of Stop&Ask):** All existing users receive this grant on cutover as a goodwill gesture, regardless of whether they've ever paid.

## Consequences

- The Explore plan provisioning lives in Sprint 7
- Cutover migration (Sprint 1) credits all existing users with a one-time bonus
- The pricing page positions Explore as "خطة الاستكشاف" — for the first journey

## Alternatives Rejected

- "First analysis free" — rejected, original problem (single-feature teaser)
- Free trial with credit card on file — rejected, friction in the GCC market
